sublime text3支持Vue语法高亮显示

发表于 2017-06-20   |   分类于 技术   |  

默认情况下，Vue.js 的单文件组件（*.vue）在 sublime 编辑器中是不被识别的。若要想高亮显示，需要安装插件 Vue Syntax Hightlight。它不仅可以使代码高亮起来，还能进行代码智能提示。那怎样安装呢？分为以下几步：

下载插件

下载地址：https://github.com/vuejs/vue-syntax-highlight解开压缩包vue-syntax-highlight-master，其内所有文件备用。

移植插件

进入Sublime，选择菜单项“Preferences->Browse Packages”，在打开的文件夹中创建Vue文件夹，然后将vue-syntax-highlight-master压缩包解压后的所有文件复制到Vue文件中。

初始化插件

按下快捷键“CTRL+SHIFT+P”，在打开的packages输入框中输入vue，选择“Set Syntax:Vue Component”进行加载。加载后重启Sublime，重新打开.vue文件，即能看到语法高亮。
