Gitbook简易教程

发表于 2017-05-03   |   分类于 技术   |  

简介

GitBook 是一个基于 Node.js 的命令行工具，可使用 Github/Git 和 Markdown 来制作精美的电子书。GitBook支持输出以下几种文档格式

  • 静态站点：GitBook默认输出该种格式
  • PDF：需要安装gitbook-pdf依赖
  • eBook：需要安装ebook-convert

安装

GitBook 是一个基于 Node.js 的命令行工具，可使用 Github/Git 和 Markdown 来制作精美的电子书。需要注意的是：用户首先需要安装 nodejs，以便能够使用 npm 来安装 gitbook。

1.npm安装Gitbook

npm install gitbook -g

2. 安装gitbook CLI脚手架工具

npm install -g gitbook-cli

3.检验下是否安装成功

gitbook -V

使用

1、创建目录，切换到目录下，执行gitbook init初始化书籍目录

gitbook init

初始化完成后，会发现目录下多了两个文件README.md和SUMMARY.md

  • README.md 是对书籍的简单介绍
  • SUMMARY.md 是书籍的目录结构

2、书籍目录结构创建完成以后,使用 gitbook serve 编译书籍

3、用浏览器打开 http://127.0.0.1:4000 查看书籍的效果

个性化配置

可以通过配置 book.json 文件来修改 gitbook 在编译书籍时的行为，例如：修改书籍的名称，显示效果等等。
